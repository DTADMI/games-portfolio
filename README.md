# React Games Portfolio

A monorepo containing web-based games built with React, Next.js, and Java Spring Boot.

## Features

- **Games**: Multiple games including Snake and Tetris
- **User Authentication**: Secure authentication with JWT
- **Leaderboards**: Global and user-specific high scores
- **Responsive Design**: Works on desktop and mobile devices
- **Docker Support**: Easy setup with Docker and Docker Compose

## Prerequisites

- Java 17+
- Node.js 16+
- Docker and Docker Compose
- PostgreSQL
- Maven
